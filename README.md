📊 Sistema de Avaliação
📝 Descrição

Programa em Python que coleta a opinião de 5 pessoas sobre um atendimento e conta quantas respostas foram Excelente, Bom ou Ruim.

⚙️ Funcionamento
Repete 5 vezes:

Pede nome e idade

Recebe a avaliação:

1 → Excelente

2 → Bom

3 → Ruim

Soma cada resposta

Mostra o total no final
